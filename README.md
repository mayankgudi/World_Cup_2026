# 2026 World Cup Predictor

A Flask-based 2026 FIFA World Cup bracket predictor.

Users can:
- Create an account
- Predict group-stage standings
- Rank third-place teams
- Generate the World Cup knockout bracket
- Pick knockout winners
- Save up to 3 brackets per account
- View other users' brackets
- Edit and delete their own brackets

## Tech Stack

- Python
- Flask
- Flask-SQLAlchemy
- Supabase PostgreSQL
- HTML
- CSS
- JavaScript
