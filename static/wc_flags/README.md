# 🏆 WC2026 Square Flags

A complete set of **48 square SVG flags** for all qualified nations of the **FIFA World Cup 2026**.

These flags were manually crafted in Adobe Illustrator — each flag was individually masked and exported in a perfect 1:1 square format, ready to use in any application or website.

---

## 🌍 Why this repo?

No clean square SVG flag set existed for the 48 WC2026 qualified teams. This repo fills that gap with high-quality, production-ready files.

---

## ✅ Features

- **48 SVG files** — one per qualified nation
- **Perfect square format** (1:1 ratio)
- **Clean & lightweight** — ready for web and mobile apps
- **No distortion** — each flag was individually masked in Illustrator
- **Free to use** — MIT License

## 🚀 Usage

Simply download the SVG file for the country you need and use it in your project:

```html
<img src="tunisie.svg" width="48" height="48" alt="Tunisia" />
```

Or inline in your code:

```jsx
import TunisiaFlag from './flags/tunisie.svg'
```
